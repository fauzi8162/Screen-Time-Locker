# 📝 Product Requirement Document (PRD) & Prompt Panduan

## 1. Ringkasan Produk (Overview)

Aplikasi Android berbasis Kotlin murni tanpa library pihak ketiga yang berfungsi sebagai *Offline Kid Timer Lock*. Aplikasi ini menerima input durasi menit dari orang tua, berjalan di latar belakang (*Foreground Service*), dan mengunci layar HP secara otomatis ketika waktu habis menggunakan API `DevicePolicyManager`.

## 2. Fitur Utama (Core Features)

* **Request Device Admin Permission:** Meminta izin hak akses administrator perangkat agar diizinkan mematikan layar secara programatis.
* **Persistent Countdown Timer:** Menghitung mundur waktu dalam satuan menit menggunakan `CountDownTimer` bawaan Android.
* **Foreground Service Protection:** Memastikan penghitung waktu tidak dimatikan paksa oleh sistem Android saat anak sedang bermain game berat atau menonton video.
* **Instant Screen Lock:** Mematikan dan mengunci layar secara instan begitu waktu habis, lalu otomatis menutup *service* latar belakang.

## 5. Konfigurasi CI/CD Otomatis via GitHub Actions

Agar GitHub bisa mem-build file `.apk` secara otomatis saat Anda melakukan `git push` dari VS Code, buat file ini di folder `.github/workflows/`:

### 📄 `.github/workflows/android-build.yml`

```yaml
name: Android CI/CD (Build APK)

on:
  push:
    branches: [ "main" ] # Memicu build tiap Anda push ke branch main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout Repository
      uses: actions/checkout@v4

    - name: Set up JDK 17
      uses: actions/setup-java@v4
      with:
        java-version: '17'
        distribution: 'zulu'
        cache: gradle

    - name: Grant Execute Permission for Gradlew
      run: chmod +x gradlew

    - name: Build Debug APK
      run: ./gradlew assembleDebug

    - name: Upload APK Result
      uses: actions/upload-artifact@v4
      with:
        name: KidLockTimer-Debug-APK
        path: app/build/outputs/apk/debug/app-debug.apk

```

## 🚀 Alur Kerja Anda di VS Code

1. **Inisialisasi Project:** Buka VS Code di folder kosong, buat file-file di atas sesuai jalurnya.
2. **Inisialisasi Git:** Buka terminal di VS Code, ketik:
```bash
git init
git add .
git commit -m "Initial Lock Screen Timer Project"

```


3. **Hubungkan ke GitHub:** Buat repositori baru di akun GitHub Anda (pilih *Private* agar kode Anda aman), lalu ikuti instruksi GitHub untuk me-remote repositori tersebut.

```bash
   git remote add origin https://github.com/username-anda/nama-repo.git
   git branch -M main
   git push -u origin main

```

4. **Unduh APK-nya:** Begitu Anda push, buka tab **Actions** di halaman GitHub Anda. Tunggu sekitar 2-3 menit sampai proses kompilasi selesai. Anda bisa langsung mengunduh file `app-debug.apk` dari sana, lalu kirim ke HP anak untuk diinstal!
